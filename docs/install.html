1. Download from Official Sources

Always download XIUM only from our official website or our GitHub release server: https://github.com/xium-ai/xium/releases. This ensures you are getting the latest and untampered version of our installer.

2. Get the Installer

Download xium.dmg and optionally xium.dmg.sig (for signature verification) from this link: xium.dmg.sig and xium.dmg You can also find it on our releases page.

3. Verify the Download (Optional)

To ensure the integrity of the download, we provide a SHA-256 checksum. Open your terminal and run: 


shasum -a 256 Downloads/xium.dmg

The output must be: be8ee6fb23b6230d7f6061ea0bb7a4ad47a9e552bda165fcfdd037301cc88315


We also sign our software using our own GPG key, because we intentionally do not use Apple‚Äôs notarization services.

To verify the signature, run:


gpg --verify Downloads/xium.dmg.sig Downloads/xium.dmg

You should see output similar to: 


gpg: Signature made Thu 29 May 17:35:52 2025 BST
gpg:                using RSA key 3DBD3594B4C2E55B5256334FB91C871A7C682D48
gpg: Good signature from "xium <info@xium.ai>" [ultimate]

4. Install and Launch

  1. Double-click the xium.dmg file to mount the installer disk image.
  2. A Finder window will open showing the XIUM app and the Applications folder.
  3. Drag the XIUM app into the Applications folder to install it.
  4. To remove macOS quarantine restrictions and allow the app to run, open the Terminal and run: 

xattr -dr com.apple.quarantine /Applications/Xium.app

You can now launch the app normally, without Apple‚Äôs signing or approval.
