Verify Our Software with GPG

  1. Download the XIUM public GPG key:
    Download XIUM Public GPG Key
  2. Install GPG:
    • macOS: GPGTools
    • Windows: Gpg4win
    • Linux: Install via your package manager (e.g. sudo apt install gnupg)
  3. Import the key into your GPG keychain:

    If you're using a graphical GPG application (e.g. GPG Keychain on macOS or Kleopatra on Windows):

    • Open the application, then use the menu: File → Import… and select xium.asc

    Alternatively, use the command line (Linux/macOS/Windows):

    gpg --import xium.asc
  4. Verify downloaded software:
    If we provide a signature file (e.g. xium.dmg.asc), run:
    gpg --verify xium.dmg.asc xium.dmg
